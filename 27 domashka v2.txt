1. Создать проект.

2. Подключить библиотеку jUnit в pom.xml к проекту.

3. Подключить плагин maven-surefire в pom.xml

4. Создать класс SimpleMathLibrary. Класс содержит два метода:
- добавление (add) для двух аргументов. Возвращается тип double.
- вычитание (minus) для двух аргументов. Возвращается тип double.

5. Создать класс SimpleMathLibraryTest. Класс содержит методы-тесты (сценарии) с аннотацией @Test

6. В классе SimpleMathLibraryTest реализовать сценарии с проверкой результатов операций:
- SimpleMathLibrary#add
- SimpleMathLibrary#minus
Проверка ожидаемого результата производится с помощью статических методов из класса Assertions.

7. Написать метод, которому в качестве аргумента передается не пустой одномерный массив.
Метод должен вернуть новый массив, полученный путем извлечения из исходного массива
идущих после последней четверки элементов. Входящий массив должен содержать хотя бы одну четверку,
иначе в методе необходимо выбросить RuntimeException.
Написать набор тестов этого метода (по 3-4 варианта входных данных).
Например: Вход - [1 2 4 4 2 3 4 1 7] -> выход: [1 7].

8. Написать метод, проверяющий состав массива из чисел 1 и 4.
Если в нем нет хоть одной четверки или единицы, то метод вернет false;
Написать набор тестов этого метода (по 3-4 варианта входных данных).
К примеру:
[ 1 1 1 4 4 1 4 4 ] -> true
[ 1 1 1 1 1 1 ] -> false
[ 4 4 4 4 ] -> false
[ 1 4 4 1 1 4 3 ] -> false
